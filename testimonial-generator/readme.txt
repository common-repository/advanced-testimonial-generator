Advanced Testimonial generator is a simple but unique plugin for adding Testimonials directely from your blog.

== Description ==

This is a flexible testimonial form plugin which generates a front end form and allow users to send their feedbacks from the site. 
It is powerful and flexible plugin than other conventional plugins which work almost in same manner. 

you can use use [Testimonials] short code in any post or page to display form and use [view data] to show latest testimonials in slider.

in admin panel, there is option to view and delete testimonials.



== Installation ==

1. Upload `testimonial.zip` to the `/wp-content/plugins/` directory
2. Unzip archive
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Use [Testimonials] shortcode to make a form for testimonials.
5. Use [view data] shortcode to show all the testimonials in a beautiful carousel.
